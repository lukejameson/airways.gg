# Flight Detail Components

This directory contains components for the flight detail page.

## Components

- `FlightHeader.svelte` - Flight title, status, airline info
- `FlightTimeline.svelte` - Departure/arrival timeline display
- `FlightMap.svelte` - Map visualization wrapper
- `WeatherDisplay.svelte` - Weather information display
- `RotationHistory.svelte` - Aircraft rotation history table
- `DelayAnalysis.svelte` - Delay reasons and analysis
