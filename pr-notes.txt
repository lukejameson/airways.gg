## PR Description

**Flight status context cards, weather accuracy fixes, and board bug fixes**

- Added delay reason cards on flight detail pages — parses Guernsey Airport status messages (weather delays, indefinite delays, holding, check-in suspended) and shows a human-readable context card with next update time where available
- Added fog/low visibility warning card on flight details when visibility at either airport is <=5km
- Fixed cancelled flights incorrectly appearing in the "done" section — they now stay visible for 3 hours after scheduled departure
- Fixed heavily delayed flights (2h+) being hidden from the board even though they hadn't departed
- Fixed weather icons showing sunny when conditions are overcast/foggy — SCT cloud (50%) now correctly maps to the cloud icon; METAR observations now take priority over TAF forecast rows
- Switched Guernsey scraper live mode to use the JSON API instead of HTML scraping
- FR24 estimated times no longer overwrite a Guernsey-sourced estimate that is still in the future

## Weather Backfill Command
# Build:
docker build -f apps/weather-backfill/Dockerfile -t weather-backfill .

# Run:
docker run --rm --network web -e DATABASE_URL=postgresql://... weather-backfill --start YYYY-MM-DD --end YYYY-MM-DD

# Specific airports only:
docker run --rm --network web -e DATABASE_URL=postgresql://... weather-backfill --start YYYY-MM-DD --end YYYY-MM-DD --airports GCI,JER
